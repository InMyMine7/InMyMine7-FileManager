## [v4.0.0] - 2026-04-24 🚀 (Latest)

### ✨ New Features
- Full rewrite with modern dark theme UI
- 15+ built-in tools and utilities
- Multi-language support (English & Indonesian)
- Responsive mobile interface
- Advanced recursive file search
- Terminal/shell integration with command history
- Bulk file operations (ZIP, delete, move, copy)
- Drag & drop file upload (multi-file)
- Built-in code editor with syntax highlighting
- List and grid view modes

### 🛠️ Tools Added
- PHP Info viewer
- Image Conversion & Resize
- Base64 Encoder/Decoder
- Hash Generator (MD5, SHA1, SHA256)
- .htpasswd Generator
- Cron Expression Builder
- Chmod Calculator
- JSON Formatter & Validator
- Regex Tester
- System Info
- UUID Generator
- URL Encoder/Decoder
- Text Statistics
- Password Generator
- Timestamp Converter

### 🔐 Security
- Session-based authentication
- Password protection
- Security documentation added (`security.md`)
- Legal disclaimer added (`disclaimer.md`)

### 🐛 Bug Fixes
- Initial stable release
